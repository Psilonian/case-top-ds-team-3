# Анализ интереса к технологическим тегам на Stack Overflow

Данный репозиторий содержит агрегированные данные для анализа интереса
к языкам программирования и технологиям на основе частоты новых вопросов
на платформе Stack Overflow.

В рамках анализа рассматриваются следующие теги:
- Python
- Java
- Javascript

Цель работы — сравнить популярность технологий, выявить динамику интереса
во времени и проанализировать сезонность по дням недели.

---

## Описание данных

Все датасеты расположены в директории `data/`.

### Доступные датасеты
- `questions_by_day.csv`  
  Дневная агрегация количества новых вопросов по тегам.


---

## Период анализа
Анализ охватывает непрерывный период продолжительностью **от 2 до 6 месяцев**
(конкретный период указывается в итоговом отчёте).

---

## Таймзона
Все временные данные приведены к таймзоне **UTC**.

---

## Источник данных
Публичные данные Stack Overflow, собранные с использованием Stack Exchange API.

---

## Назначение данных
Данные используются для:
- анализа трендов
- сравнения популярности технологий
- анализа сезонности по дням недели
- визуализации в BI-инструментах (например, Yandex Datalens)
