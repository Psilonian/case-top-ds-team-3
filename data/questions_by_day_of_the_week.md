# Data Contract: questions_by_weekday.csv

## Назначение
Датасет содержит агрегированные данные по дням недели и используется
для анализа сезонности активности вопросов по технологическим тегам.

---

## Временная гранулярность
День недели (Monday–Sunday).

---

## Таймзона
Данные получены на основе дневной агрегации, выполненной в **UTC**.

---

## Схема данных

| Поле | Тип | Описание |
|------|-----|----------|
| `weekday` | string | День недели |
| `tag` | string | Технологический тег |
| `avg_questions_count` | float | Среднее количество вопросов |

---

## Допустимые значения

### `weekday`
Допустимые значения:
- Monday
- Tuesday
- Wednesday
- Thursday
- Friday
- Saturday
- Sunday

### `tag`
Допустимые значения:
- `Python`
- `Java`
- `Javascript`

---

## Правила агрегации
- Датасет является производным от `questions_by_day.csv`.
- Для каждой пары (`weekday`, `tag`) рассчитывается среднее значение
  `questions_count`.
- Используется среднее значение, а не сумма, чтобы компенсировать
  различное количество дней недели в анализируемом периоде.

---

## Гарантии качества данных
- Отсутствуют пропущенные значения.
- Каждая комбинация (`weekday`, `tag`) представлена ровно один раз.
- Поле `avg_questions_count` является неотрицательным числом с плавающей точкой.
- Кодировка CSV: UTF-8.
- Разделитель столбцов: запятая (`,`).

---

## Пример данных

```csv
weekday,tag,avg_questions_count
Monday,Python,145.2
Monday,Java,102.8
Monday,Javascript,118.5
```
