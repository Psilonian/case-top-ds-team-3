# Сбор данных Stack Overflow

## Описание
В рамках задания реализован автоматизированный сбор данных с платформы Stack Overflow
с использованием официального Stack Exchange API (v2.3).

Данные используются для оценки интереса к технологиям по частоте задаваемых вопросов
с разными тегами.

---

## Источник данных
Официальный API Stack Exchange:  
https://api.stackexchange.com

Используемый endpoint:
- `/2.3/questions`

---

## Способ получения данных
Сбор данных выполняется программно с помощью Python и библиотеки `requests`.

Для каждого заданного тега выполняются HTTP-запросы к API с фильтрацией:
- по тегу
- по диапазону дат (2–6 месяцев назад от текущей даты)
- по дате создания вопроса

Учитываются ограничения API (параметр `backoff`).

---

## Обрабатываемые теги
- python
- java
- javascript

---

## Формат выходных данных
Результат сохраняется в CSV-файл со следующими полями:

| Поле | Описание |
|-----|----------|
| `date` | Дата (день или начало недели, UTC) |
| `tag` | Тег вопроса |
| `questions_count` | Количество вопросов |

---

## Агрегация
Поддерживается агрегация:
- по дням (`day`)
- по неделям (`week`, ISO-неделя, понедельник)

---

## Запуск

```bash
python scripts/fetch_data.py \
  --tags python java javascript \
  --grain day \
  --out data/questions_by_date.csv
